# Endstone Essentials Plugin

Plugin para Endstone que añade funcionalidades clásicas estilo *Essentials* como `/home`, `/tpa` y más utilidades pensadas para mejorar la experiencia en servidores.

## Características

* Sistema de homes (`/sethome`, `/home`, `/delhome`)
* Teletransportación entre jugadores (`/tpa`, `/tpaccept`, `/tpdeny`)
* Comandos básicos de utilidad
* Diseño ligero y fácil de integrar
* Configuración sencilla

## Comandos

| Comando     | Descripción                          |
| ----------- | ------------------------------------ |
| /sethome    | Establece tu hogar                   |
| /home       | Te teletransporta a tu hogar         |
| /delhome    | Elimina tu hogar                     |
| /tpa <user> | Solicita teletransporte a un jugador |
| /tpaccept   | Acepta una solicitud de tpa          |
| /tpdeny     | Rechaza una solicitud de tpa         |

## Instalación

1. Descarga el plugin
2. Coloca el archivo en la carpeta `plugins/` de tu servidor Endstone
3. Reinicia el servidor

## Configuración

El plugin generará automáticamente un archivo de configuración en la carpeta correspondiente. Puedes modificarlo según tus necesidades.

## Requisitos

* Servidor compatible con Endstone
* Versión de Java requerida por el servidor
